# CSE321 Lab Project – Process Management Simulation

This repository contains my **CSE321 (Operating Systems Lab)** project from **BRAC University**.
The project is designed to simulate key concepts of **process management and synchronization** in an operating system.


#📌 Project Overview

The goal of this project is to demonstrate how processes interact and are controlled within a system.
It focuses on understanding real-world OS behavior through a simple simulation.

The project covers:

* Process execution flow
* Synchronization using **mutex**
* Communication using **condition variables**
* Waiting and signaling mechanisms (`pm_wait`)
* Basic handling of concurrent processes


#🛠️ Technologies Used

* Programming Language: C
* Core Concepts: Operating Systems (Process Management, Synchronization)
* Tools: GCC Compiler
